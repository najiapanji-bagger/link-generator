File ini adalah website kedua untuk generator link tamu.

Cara pakai:
1. Upload file index.html ini ke project / repo baru.
2. Deploy ke Vercel.
3. Buka websitenya.
4. Isi nama tamu, misalnya: Doni
5. Hasil link akan otomatis menjadi:
   https://walimatussafar-invitation.vercel.app/?to=Doni

Catatan:
- Link utama undangan sudah dipasang:
  https://walimatussafar-invitation.vercel.app/
- Anda tidak perlu edit manual lagi.
